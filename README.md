# Sistema POS "Tía Pan" - Proyecto de Análisis y Diseño de Sistemas

Este repositorio contiene la solución informática desarrollada para la panadería y tienda "Tía Pan", ubicada en Ochoa León (Cuenca). El sistema reemplaza la libreta y calculadora manual por un POS digital.

## Requerimientos Cumplidos
- **Arquitectura:** Aplicación Web Multiusuario diseñada con Vite, React y Tailwind CSS.
- **Módulos:**
  - Login de Usuarios (Dueña / Cajeros).
  - POS (Caja Rápida) para ventas de pan y abarrotes.
  - Gestión de Pedidos Especiales.
  - Inventario de Tienda.
  - Reporte y Cuadre de Caja (Z).
- **Modelo Orientado a Objetos:** El código implementa las clases del diagrama estructural (`Producto`, `Panaderia`, `AbarroteTienda`, `Venta`).

## Instalación
1. `npm install`
2. `npm run dev`
